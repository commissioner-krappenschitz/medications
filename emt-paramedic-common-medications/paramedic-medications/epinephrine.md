# Epinephrine

Epinephrine is given for anaphylaxis and cardiac arrest. There are two concentrations for this medications.

{% hint style="info" %}
**1:1,000** is for **anaphylaxis** and is administered **0.3 - 0.5 mg** IM every 5 minutes as needed for a maximum of 2mg.
{% endhint %}

{% hint style="info" %}
**1:10,000** is for **cardiac arrest** and is administered **1 mg** every 3 - 5 minutes as needed for a maximum of 6 mg.
{% endhint %}
